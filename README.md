Student Feedback Analysis

This project analyzes student feedback collected from college events such as workshops, technical fests, and cultural programs. The goal is to extract meaningful insights from both rating-based and text-based feedback using data analysis and Natural Language Processing (NLP) techniques.
Tasks Performed:

Cleaned and prepared feedback dataset (Google Form CSV export).

Analyzed satisfaction ratings (1–5 scale).

Performed sentiment analysis using TextBlob / VADER

Visualized key insights through bar charts, pie charts, and word clouds.

Interpreted results to recommend improvements for future events.

Skills Gained:

Data cleaning and transformation using pandas

Sentiment analysis using NLP tools (TextBlob/VADER)

Visualization using matplotlib and seaborn

Insight generation and interpretation from survey data

Tools & Libraries Used:

Google Colab – Cloud-based notebook environment

pandas – Data cleaning and manipulation

matplotlib / seaborn – Visualization

TextBlob / VADER – Sentiment analysis

wordcloud – Text visualization
